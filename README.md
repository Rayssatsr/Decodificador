The following rules are used to encode the text:

- `e` becomes `enter`
- `i` becomes `imes`
- `a` becomes `ai`
- `o` becomes `ober`
- `u` becomes `ufat`

For example, the word `gato` would be encoded as `gaitober`, and `gaitober` would be decoded back to `gato`.
